## 🚀 NetPulse — macOS Network Speed Monitor + Speed Test

NetPulse is a lightweight macOS menu bar app that shows your real-time internet speed directly in the status bar — now enhanced with a built-in Speed Test for measuring your actual network performance.

It’s fast, minimal, and designed to feel native — no clutter, no heavy UI, just clean and powerful network monitoring.


## ✨ Features

📊 Real-Time Monitoring
	•	Live Speed in Menu Bar
Instantly view download & upload speed
	•	⚡ Smart Units (Auto KB/s ↔ MB/s)
Automatically adjusts based on speed
	•	📦 Total Data Usage
Tracks cumulative download & upload


## 🚀 Speed Test (NEW)
	•	Measure actual internet performance (Download & Upload)
	•	One-click test directly from menu
	•	Clean and minimal results display
	•	Works alongside live monitoring


## 🌐 Network Awareness
	•	Detects active connection (WiFi / Ethernet)


## 🎛 Controls
	•	Toggle monitoring ON/OFF
	•	Reset usage anytime
	•	Run Speed Test on demand


## 🍎 Native macOS Experience
	•	Built using SwiftUI + MenuBarExtra
	•	Lightweight and efficient
	•	Designed for seamless background usage


## 📦 Installation
	1.	Download NetPulse.dmg from Releases
	2.	Open the DMG
	3.	Drag NetPulse.app → Applications
	4.	Launch the app

⚠️ macOS Security Warning

Since the app is not signed:
	•	Right click → Open
	•	Click Open again



## 🧠 Tech Stack
	•	SwiftUI
	•	Combine
	•	MenuBarExtra
	•	getifaddrs() (low-level network stats)
	•	URLSession / Network framework (for Speed Test)
