# 🚀 NetPulse — macOS Network Speed Monitor

NetPulse is a lightweight macOS menu bar app that shows your **real-time internet speed** directly in the status bar.

It’s fast, minimal, and designed to feel native — no clutter, no heavy UI, just clean network monitoring.

---

## ✨ Features

- 📊 **Live Speed in Menu Bar**  
  See download & upload speed instantly

- ⚡ **Smart Units (Auto KB/s ↔ MB/s)**  
  Automatically adjusts based on speed

- 📦 **Total Data Usage**  
  Tracks total download & upload

- 🌐 **Network Detection**  
  Shows WiFi or Ethernet

- 🎛 **Controls**
  - Toggle monitoring ON/OFF  
  - Reset usage anytime  

- 🍎 **Native macOS UI**  
  Built using SwiftUI + MenuBarExtra




  ---

## 📦 Installation

1. Download `NetPulse.dmg` from Releases  
2. Open the DMG  
3. Drag **NetPulse.app** → Applications  
4. Launch the app  

---

## ⚠️ macOS Security Warning

Since the app is not signed:

- Right click → Open  
- Click **Open** again  

---

## 🧠 Tech Stack

- SwiftUI  
- Combine  
- MenuBarExtra  
- `getifaddrs()` (low-level network stats)
